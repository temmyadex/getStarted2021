How to setup the Personal Access Token(PAT)

  1. First click on your profile picture
  2. Next navigate to the setting section of the dropdown
  3. Take note of the navigation by the left hand side, scroll to the very end of that part, then click on Developer Setting
  4. Next click on the Generate new token
  5. After this take time to go through all the checklist and ensure to set the duration for which you need this token for, for security reason, it is advisable and recommended

Cloning existing repository from GitHub

  • Locate the repo on GitHub or better still create a new repo
  • Ensure all is set, then clone the repo to your local machine
  • To clone this, execute this command: "git clone REPO-URL-FROM-GITHUB" then hit the Return/Enter Key
  • By so doing, you have been able to make a copy of the repo on your local machine
  • create all your file structure as might required by your project
  • Follow with this git command to ensure and check the status, "git status"
