This is a simple step documentation of the newly updated GitHub Authentication utilising the Personal Access Token

How to setup the Personal Access Token(PAT)

  1. First click on your profile picture
  2. Next navigate to the setting section of the dropdown
  3. Take note of the navigation by the left hand side, scroll to the very end of that part, then click on Developer Setting
  4. Next click on the Generate new token
  5. Do ensure to copy this toekn somewhere for easy access when it is needed and for the duration it was specified for
  6. After this take time to go through all the checklist and ensure to set the duration for which you need this token for, for security reason, it is advisable and recommended
  7. Scrolled to the end for the final stage, by clicking on Create/Generate as might indicate at the botrtom of the page
